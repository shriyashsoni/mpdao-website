# MP DAO Brand Kit

Welcome to the official MP DAO brand kit. This repository contains all the assets, guidelines, and resources needed to maintain consistent branding across our community initiatives.

## Contents

### Logo Variations

- **mpdao-logo-black.png** - Solid black logo for light backgrounds
- **mpdao-logo-outline.png** - Transparent outline version for dark backgrounds
- **mpdao-logo-light.png** - Light outline on light background
- **mpdao-logo-dark.png** - Bold dark logo on white background

## Color Palette

### Primary Colors
- **Black**: #000000 - Primary brand color
- **White**: #FFFFFF - Supporting color

### Accent Colors
- **Blue**: #3B82F6 (rgb(59, 130, 246)) - Primary accent
- **Purple**: #8B5CF6 (rgb(139, 92, 246)) - Secondary accent

## Typography

### Font Families
- **Headings**: Geist (Weights: 700, 600, 500)
- **Body Text**: Geist (Weights: 400, 500)
- **Monospace**: Geist Mono (Weight: 400)

## Usage Guidelines

### Do's ✓
- Use logos with appropriate spacing and clearance
- Maintain aspect ratio when resizing logos
- Use black logo on light backgrounds
- Use white/outline logo on dark backgrounds
- Apply consistent branding across all materials
- Follow the color palette strictly

### Don'ts ✗
- Don't distort or skew the logo
- Don't use colors outside the provided palette
- Don't rotate or flip the logo
- Don't place logo on conflicting backgrounds
- Don't modify the logo shape or proportions
- Don't change the font family or weights

## Logo Usage by Context

| Context | Logo | Usage |
|---------|------|-------|
| Light Background | Black Logo | Websites, presentations, documents |
| Dark Background | Outline Logo | Dark backgrounds, overlays, dark UI |
| Social Media | Black or Outline | Depends on background color |
| Print | Dark Logo | Business cards, flyers, banners |
| Digital | Black or Outline | Web graphics, social media graphics |

## Minimum Logo Size

For optimal clarity and recognition:
- **Digital**: Minimum 100px width
- **Print**: Minimum 1 inch (72mm) width

## Clear Space

Always maintain clear space around the logo. The minimum clear space is equal to the height of the "M" in the logo.

## Accessibility

- All logos should have sufficient contrast ratio (minimum 4.5:1)
- Always include alt text: "MP DAO logo"
- Use semantic HTML markup for branding elements

## Social Media Specifications

### Profile Pictures
- **Recommended Size**: 400x400px
- **Format**: PNG with transparent background
- **Use**: Black outline logo

### Header Images
- **Facebook**: 820x312px
